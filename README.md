# Did You Pay? website

The static public website for Did You Pay?, built for hosting on GitHub Pages.

## Technology

- HTML5
- CSS3
- Vanilla JavaScript

The project intentionally uses no frameworks or build tools. Use a local static web server to preview the complete site, including the dynamically loaded supported-locations data.

## Pages

- Home
- Frequently asked questions
- Supported locations
- Privacy
- Terms of use
- Contact
- Release notes
- Custom 404 page

Shared presentation and behaviour live in `css/style.css` and `js/script.js`. Approved content is recorded in `CONTENT_GUIDE.md`, design guidance lives in `DESIGN_SYSTEM.md`, supported locations are maintained in `data/locations-web.json`, and published branding assets live in `images/branding`.

The website includes approved branding, responsive image derivatives, browser icons, automatic light and dark themes, privacy and legal pages, support information, release notes, and a custom 404 page.

## GitHub Pages

Configure GitHub Pages to publish from the repository's default branch and root directory. No build step is required.

## Before launch

Replace the App Store placeholder link with the approved production URL and update the Version 1.0.0 release date when the app is released.
